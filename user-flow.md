# User Flow - SisterSignal

## Flow Description

1. User opens the application
2. User sees the main interface with a "Request Help" option
3. User clicks "Request Help"
4. User selects the type of help needed:
   - Escort
   - Pickup
   - Check-in
5. User's location is automatically detected
6. Request is sent to nearby verified users
7. Nearby users receive a notification
8. A user accepts the request
9. A connection is established between requester and responder
10. Help process is completed
