# Features Folder

This repository contains the working source code for SisterSignal.

The project was developed and synced through Lovable, so the source code follows Lovable’s generated repository structure rather than a manually reorganized `features/` layout.

Main source files are kept in their original locations in order to preserve deployment stability and keep the live prototype functional.
